##How add package to your project?

###Maven
```xml
<dependency>
  <groupId>io.github.vladshyrokyi</groupId>
  <artifactId>template-message-sdk</artifactId>
  <version>1.0</version>
</dependency>
```

###Gradle
```groovy
repositories {
    mavenCentral()
}
dependencies {
    compile 'io.github.vladshyrokyi:template-message-sdk:1.0'
}
```
